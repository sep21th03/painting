Introduction

API DOCS

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Lưu đánh giá sản phẩm.

Phương thức này nhận yêu cầu chứa thông tin đánh giá sản phẩm từ người dùng, kiểm tra và xác thực dữ liệu đầu vào bằng lớp StoreReviewRequest, sau đó chuyển dữ liệu đã xác thực cho ProductService để lưu đánh giá.

Example request:
curl --request POST \
    "http://localhost/api/product-detail/review" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"product_id\": \"et\",
    \"rating\": 1,
    \"comment\": \"yfwsigfqyv\"
}"

const url = new URL(
    "http://localhost/api/product-detail/review"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "product_id": "et",
    "rating": 1,
    "comment": "yfwsigfqyv"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Store a new order in the database.

This method handles the creation of an order, including the generation of the order code, storing order details, updating product stock, and removing items from the cart. It also dispatches a job to notify the user of the order status.
